                          UmeFont for TCPDF

This package includes fonts for TCPDF, based on UmeFont.

UmeFont: http://sourceforge.jp/projects/ume-font/

= Installation =

1.Upload all *.php, *.z, *.ctg.z files to /tcpdf/fonts/ directory.
2.Set as defaut font using setHeaderFont(), setFooterFont(), SetDefaultMonospacedFont, SetFont() etc.
3.If you allow persons who receive a PDF to make changes to it even if they don't have the same font, set "setFontSubsetting" to "false" using setFontSubsetting().

= インストール =

1.*.php、*.z、*.ctg.zという名前のファイルを全て、「/tcpdf/fonts/」ディレクトリにアップロードします。
2.setHeaderFont()、setFooterFont()、SetDefaultMonospacedFont、SetFont()などを使って、デフォルトのフォントとして設定します。
3.PDFを受け取った相手が、同じフォント持っていなくとも編集出来るようにするには、setFontSubsetting()を使って「setFontSubsetting」を「false」に設定します。

各フォントの名前は次の通りです。
